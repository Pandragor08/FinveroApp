# App Android Finvero
Automatización de App finvero con Appium, Selenium, TestNG y Allure

## Características

- Validacion seccion Score
- Validar seccion inicio de sesion
- Validacion Home
- Validar más
- Validar banners
- Generación de reporteria

  ## Tecnologías utilizadas

- Java 
- IDE Intellij
- Appium
- Selenium
- TestNG
- Appium inspector
